# User Explorer App

A simple web app that fetches and displays random users from the Random User API.

## Features

- Load 12 random users
- Filter users by gender (male/female)
- Sort users alphabetically by name
- Responsive user cards with photos

## Files

- `index.html` - Main HTML structure
- `app.js` - JavaScript functionality
- `style.css` - Styling (if present)

## Usage

1. Open `index.html` in a web browser
2. Click "Load Users" to fetch random users
3. Use filter buttons to show only male or female users
4. Click "Sort by Name" to alphabetically sort users

## Troubleshooting

If names aren't displaying correctly, open browser console (F12) and check for API response structure when clicking "Load Users".
